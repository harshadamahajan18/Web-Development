CSS is used to style and layout web pages. It allows you to apply styles to HTML elements, control their appearance, and manage the layout of a webpage
Key CSS Concepts:
Selectors: Determine which HTML elements are affected by the styles.

  • Element Selector: Selects all elements of a given type. p { color: blue; }
  • Class Selector: Selects all elements with a specific class. .highlight { background-color: yellow; }
  • ID Selector: Selects a single element with a specific ID. #main { max-width: 800px; }
  • Attribute Selector: Selects elements based on an attribute or attribute value a[target="_blank"] { color: red; }
